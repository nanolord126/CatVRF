🔄 HOW IT WORKS
─────────────────────────────────────────────────────────────────────────────

  Every 30 Seconds:

    ┌─ Browser Timer (30s) ─────────────────────────┐
    │                                                │
    │  0:00  wire:poll.30000ms triggered           │
    │  0:00  Livewire.pollChartData() called        │
    │  0:00  Check: isLoading? (skip if true)       │
    │  0:01  HTTP GET /api/analytics/heatmaps/*     │
    │  0:02  ClickHouse query executed              │
    │  0:03  Response formatted + data updated      │
    │  0:04  Chart.js re-renders                    │
    │  0:04  Log: "Poll completed successfully"     │
    │        ⏰ Wait 30 seconds                      │
    │        (Repeat)                               │
    │                                                │
    └────────────────────────────────────────────────┘

  If WebSocket Event Arrives Earlier:

    ┌─ Real-Time Update ────────────────────────────┐
    │                                                │
    │  [Before 30s poll]                            │
    │  ClickHouse data changes                      │
    │         ↓                                      │
    │  Sync job completes                           │
    │         ↓                                      │
    │  Broadcast event                              │
    │         ↓                                      │
    │  Echo.js listener triggers                    │
    │         ↓                                      │
    │  Chart updates IMMEDIATELY ✨                 │
    │  (No waiting for 30s poll)                    │
    │                                                │
    └────────────────────────────────────────────────┘
