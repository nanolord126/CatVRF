## 🧪 ЧТО ТЕСТИРОВАТЬ

### На Desktop (Windows/Mac):

```
1. ВРАЩЕНИЕ (Rotation)
   - Нажмите и тащите мышь влево-вправо
   - Модель должна вращаться по оси Y

2. МАСШТАБИРОВАНИЕ (Zoom)
   - Крутите колесо мыши вверх/вниз
   - Модель должна приближаться/отдаляться (0.5x - 3.0x)

3. ЦВЕТА (Color Selection)
   - Нажимайте кнопки цветов (если они есть)
   - Материалы должны меняться (золото, серебро, платина)

4. НАВИГАЦИЯ
   - Скролл страницы вверх/вниз
   - Все 6 карточек должны видеть
   - Нажмите на каждую карточку
```

### На Mobile (iPhone/Android):

```
1. СВАЙП ВРАЩЕНИЕ (Swipe Rotation)
   - Проведите пальцем влево-вправо
   - Модель должна вращаться

2. PINCH МАСШТАБИРОВАНИЕ (Pinch Zoom)
   - Разведите два пальца (zoom in)
   - Сожмите два пальца (zoom out)

3. ПОВОРОТ УСТРОЙСТВА (Device Orientation)
   - Поверните телефон
   - AR режим должен активироваться (если HTTPS)

4. ПОЛНЫЙ ЭКРАН (Fullscreen)
   - Двойной тап может активировать полный экран
```
