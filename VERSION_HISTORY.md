## 📝 ПРИМЕЧАНИЯ

### Важная информация

⚠️ **НДС 18% больше не поддерживается** (не существует с 2019 года)
- Все коды VAT_18 → VAT_20
- Все Vat18 → Vat20
- Все vat18 → vat20

✅ **Поддерживаемые ставки НДС**:
- 0% (льготная ставка)
- 10% (льготная ставка)
- 20% (стандартная ставка с 2019 года)
- Без НДС (УСН, ЕСХН, ЕНВД, ПСН)

### Совместимость

- **Laravel**: 11+ (использует Illuminate facades)
- **PHP**: 8.0+
- **API Atol**: v2.5
- **API CloudKassir**: REST (latest)
- **API Tinkoff**: v2.0+

### Производительность

- Все операции используют асинхронный HTTP клиент (Illuminate\Support\Facades\Http)
- Timeouts установлены: 10-15 секунд для разных операций
- Логирование структурировано для audit trail
