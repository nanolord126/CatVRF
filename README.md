# CatVRF - Canon Automated Testing & Verification Runtime Framework

CatVRF - это фреймворк для автоматизированного тестирования и верификации, разработанный для обеспечения соответствия кода канону проекта. Он выполняет статический анализ, проверяет архитектурные правила и гарантирует, что код соответствует высоким стандартам качества, необходимым для крупномасштабных приложений.

## Основные возможности

- **9-слойная архитектура**: Строгое соблюдение архитектурных слоев для всех вертикалей.
- **AI-конструкторы**: Обязательное наличие AI-конструкторов для каждой из 127 вертикалей.
- **B2C/B2B**: Разделение логики для B2C и B2B сценариев.
- **Fraud Control**: Интегрированная система контроля мошенничества.
- **Multi-tenancy**: Поддержка множества тенантов с изоляцией данных.
- **Real-time Geotracking**: Геотрекинг в реальном времени для сервисов доставки.
- **И многое другое**: Продвинутая система управления корзиной, кошельками, бонусами и т.д.

## Технологический стек

- **Backend**: Laravel 11.x (PHP 8.3+)
- **Frontend**: Livewire 3, Alpine.js, Tailwind 4, Vue 3
- **Базы данных**: PostgreSQL 16, ClickHouse, Redis 7+
- **Инфраструктура**: Docker, Laravel Sail, S3-совместимое хранилище

## Установка

1.  Склонируйте репозиторий.
2.  Установите зависимости: `composer install`
3.  Настройте файл `.env`.
4.  Запустите миграции: `php artisan migrate`
5.  Запустите проект: `php artisan serve`
