🚀 DEPLOYMENT STEPS
═════════════════════════════════════════════════════════════════════

1. REGISTER SERVICE PROVIDER
   File: config/app.php
   Add: \App\Providers\MLServiceProvider::class,

2. RUN MIGRATIONS
   Command: php artisan migrate

3. PUBLISH CONFIGURATION
   Command: php artisan vendor:publish --tag=taste-ml-config

4. CONFIGURE ENVIRONMENT
   File: .env
   Set: OPENAI_API_KEY=sk-proj-...
   Set: REDIS_HOST=127.0.0.1
   Set: REDIS_PORT=6379

5. REGISTER SCHEDULER
   File: app/Console/Kernel.php
   Add in schedule() method:
   $schedule->job(\App\Jobs\ML\MLRecalculateUserTastesJob::class)
       ->dailyAt('03:00')->timezone('UTC')->onOneServer();

6. START QUEUE WORKER
   Command: php artisan queue:work

7. RUN TESTS
   Command: php artisan test tests/Feature/ML/
   Command: php artisan test tests/Unit/ML/

8. VERIFY INSTALLATION
   Command: php artisan tinker
   Test: app(\App\Services\ML\UserTasteProfileService::class)
